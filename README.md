## 使用

```js
import Dialog from 'milkui-dialog';

<Dialog></Dialog>
```

## 参数

| Properties | Type | Description | Default |
| -- | -- | -- | -- |
| className | String | 追加的 class 类名 | '' |
| title | String | 对话框标题 | '' |
| show | Boolean | 是否显示 | false |
| buttons | Array | 定义操作按钮的数组 | [] |

## 开发

### install

```bash
cnpm i milk-dev -g    # 组件开发工具
cnpm install
npm start
```
